# Altverse / AU Reader V2

This project is a comprehensive reading platform application that connects authors and readers. It features a modern, gamified experience designed to enhance user engagement.

## Project Structure

This repository is divided into two main components:

- **frontend**: A cross-platform mobile application built with Flutter (`au_reader`). It provides an intuitive interface for users to discover, read, and interact with content, as well as an author dashboard for managing publications.
- **backend**: A robust RESTful API built with Laravel 12. It serves as the core system handling authentication, content delivery, user interactions, notifications, and gamification mechanics.

## Key Features

- **Reading & Authoring**: dedicated interfaces for reading stories and a comprehensive system for authors to publish and manage their work.
- **Gamification**: An integrated system featuring missions, level progression, and seasonal ranks to reward user activity.
- **User Engagement**: Support for comments, liking comments, bookmarks, and reading history tracking.
- **Monetization & Wallet**: A built-in wallet system handling transactions and rewards.

## Technologies

- **Frontend**: Flutter, Provider (State Management), GoRouter (Navigation).
- **Backend**: PHP 8.2, Laravel 12, MySQL (via Laravel models/migrations).
